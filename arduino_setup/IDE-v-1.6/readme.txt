
Arduino-IDE integration:
========================

1) LINUX

* Copy the folder 'madworm' to the 'hardware' folder:

  "./arduino-1.6.x/hardware"

* Start / restart the IDE.


2) Winblows

* Download and install the IDE from "www.arduino.cc"

* Start the IDE once & close it again

  An 'Arduino' folder will have been created in 'My Documents'.

* Create a folder named 'hardware' in there and copy the 'madworm'
  folder into it.

* Start the IDE again. The ISML will shop up in the list of boards.
